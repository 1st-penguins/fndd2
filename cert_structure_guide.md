# 자격증 구조 가이드

## 📋 개요
퍼스트펭귄은 이제 **두 가지 자격증**을 지원합니다:
1. 🏋️ **건강운동관리사**
2. ⚽ **2급 스포츠지도사**

## 🗂️ 폴더 구조

### 건강운동관리사 (기존)
```
├── subjects/              # 과목별 페이지
│   ├── subject_운동생리학.html
│   ├── subject_건강체력평가.html
│   └── ... (8개 과목)
├── years/                 # 연도별 페이지
│   ├── year_2024.html
│   └── ... (2019~2025)
├── exam/                  # 문제 페이지
│   ├── 2024_운동생리학.html
│   └── ... (많은 HTML 파일들)
├── data/                  # JSON 데이터
│   ├── 2024_운동생리학.json
│   └── ... (많은 JSON 파일들)
└── images/                # 문제 이미지
    ├── 2024 운동생리학/
    └── ... (많은 폴더들)
```

### 2급 스포츠지도사 (신규) ✨
```
├── subjects-sports/       # 과목별 페이지
│   └── subject_스포츠사회학.html (샘플)
├── years-sports/          # 연도별 페이지
│   └── year_2024.html (샘플)
├── exam-sports/           # 문제 페이지 (추가 예정)
├── data-sports/           # JSON 데이터 (추가 예정)
└── images-sports/         # 문제 이미지 (추가 예정)
```

## 🚀 사용 방법

### 1. 메인 페이지 (index.html)
- 상단에 자격증 선택 버튼이 있습니다
- 🏋️ **건강운동관리사** / ⚽ **2급 스포츠지도사** 클릭하면 전환됩니다
- 각 자격증별로 과목별/연도별/모의고사 탭이 독립적으로 동작합니다

### 2. 강의 페이지 (lectures.html)
- 📄 **PDF 자료** / 🎥 **인터넷강의** 탭으로 구분됩니다
- 현재 PDF 자료만 활성화되어 있습니다
- 인터넷강의는 "준비중" 상태입니다

## 📂 데이터 추가 방법

### 2급 스포츠지도사 데이터 추가하기

#### 1단계: JSON 데이터 생성
`data-sports/` 폴더에 JSON 파일 생성:
```
data-sports/2024_스포츠사회학.json
data-sports/2024_스포츠윤리.json
...
```

**JSON 파일 형식** (기존 `data/` 폴더 파일과 동일):
```json
{
  "year": "2024",
  "subject": "스포츠사회학",
  "questions": [
    {
      "questionNumber": 1,
      "questionImage": "images-sports/2024 스포츠사회학/문제 (1).png",
      "answer": "1",
      "explanation": "해설 내용..."
    }
  ]
}
```

#### 2단계: 이미지 파일 추가
`images-sports/` 폴더에 이미지 추가:
```
images-sports/2024 스포츠사회학/문제 (1).png
images-sports/2024 스포츠사회학/문제 (2).png
...
```

**권장 이미지 크기:**
- 문제 이미지: 800 x 1200 px (가로 x 세로)
- 강의 카드 썸네일: 320 x 200 px

#### 3단계: 문제 페이지 생성
`exam-sports/` 폴더에 HTML 파일 생성:
```
exam-sports/2024_스포츠사회학.html
```

**파일 생성 방법:**
- 기존 `exam/2024_운동생리학.html` 파일을 복사
- 경로 수정: `../data/` → `../data-sports/`
- 타이틀 수정: "건강운동관리사" → "2급 스포츠지도사"

#### 4단계: 과목/연도 페이지 추가
- `subjects-sports/` 폴더: 샘플 파일 복사하여 다른 과목 추가
- `years-sports/` 폴더: 샘플 파일 복사하여 다른 연도 추가

## 📊 학습 분석 (Analytics)
- 현재 학습 분석은 자격증 구분 없이 모든 데이터를 표시합니다
- 추후 자격증별로 분리하려면 `analytics.html` 수정이 필요합니다

## ⚠️ 주의사항

1. **파일명 규칙 준수**
   - 연도_과목명.json (예: `2024_스포츠사회학.json`)
   - 연도_과목명.html (예: `2024_스포츠사회학.html`)

2. **이미지 경로**
   - JSON에서: `"images-sports/2024 스포츠사회학/문제 (1).png"`
   - 건강운동관리사는 `"images/..."`
   - 2급 스포츠지도사는 `"images-sports/..."`

3. **URL 인코딩**
   - 파일명에 한글이 있으면 자동으로 URL 인코딩됩니다
   - 공백은 `%20`으로 변환됩니다

4. **기존 파일 보호**
   - 건강운동관리사 관련 파일(`subjects/`, `years/`, `exam/`, `data/`, `images/`)은 수정하지 마세요
   - 새 자격증 관련 파일만 `-sports` 접미사가 붙은 폴더에 추가하세요
