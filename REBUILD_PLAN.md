## 3. 디자인 시스템 (애플 스타일)

### 컬러
- 메인: #1D2F4E (네이비)
- 포인트/CTA: #F4511E (주황, 펭귄 부리색)
- 배경: 화이트 / 라이트그레이
- 텍스트: #1D1D1F (애플 블랙)

### 폰트
- Pretendard Variable
- 크기 스케일: 12/14/17/20/24/32/48/72px
- 본문 기본: 17px (애플 기준)
- 굵기: 400(본문) / 500(강조) / 600(제목)

### 버튼
- Primary: 네이비 배경 + 흰 텍스트, pill shape
- Secondary: 주황 배경 + 흰 텍스트
- Ghost: 테두리만, 투명 배경
- 크기: sm(32px) / md(40px) / lg(48px)
- 호버: 살짝 어둡게 + translateY(-1px)

### 카드
- 배경: 흰색, border-radius: 18px
- 그림자: 0 2px 12px rgba(0,0,0,0.08)
- 호버: 그림자 강화 + translateY(-4px)
- 패딩: 24px
- 테두리: 없음 (그림자로만 구분)

### 레이아웃
- 최대 너비: 1200px, 가운데 정렬
- 섹션 여백: 80px (모바일 48px)
- 컨텐츠 간격: 8px 그리드 기반
- 반응형: 모바일 768px 기준

### 타이포그래피 원칙 (애플 스타일)
색 대신 굵기와 크기로 위계를 만든다.

- 메인 타이틀: 700, #1D1D1F
- 서브 타이틀: 600, #1D1D1F
- 본문 강조: 500, #1D1D1F
- 본문: 400, #1D1D1F
- 보조 텍스트: 400, #6E6E73 (애플 그레이)
- 캡션/힌트: 300, #86868B (라이트 그레이)

### 컬러 사용 원칙 (미니멀리즘)
- 기본: 블랙 + 화이트 + 그레이로 해결
- 네이비 #1D2F4E: 브랜드 요소 최소한으로
- 주황 #F4511E: 페이지당 CTA 1개에만.
  절대 남발 금지.
- 색 쓰고 싶을 때 먼저 그레이로
  해결되는지 확인할 것

### 인터랙션
- 모든 트랜지션: 0.2s ease
- 호버시 미세하게 올라오는 효과
- 탭/선택 활성화: 네이비 언더라인 또는 배경
- 포커스: 주황 outline

### 컴포넌트 목록 (모든 페이지 동일하게 적용)
- 헤더: 고정, 스크롤시 blur 배경
- 푸터: 심플, 한 줄
- 탭: sticky, 스크롤시 헤더 아래 고정
- 카드: 과목/연도/문제 목록
- 토글: 과목별/연도별 전환
- 버튼: CTA, 네비게이션
- 뱃지: 신규/완료/강의있음 표시
- 프로그레스바: 학습 진행률
- 모달: 로그인
