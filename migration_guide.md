# Linear Design System 홈페이지 마이그레이션 가이드

## 🎉 새로운 Linear 디자인 시스템 적용 완료!

Linear.app에서 추출한 디자인 시스템을 기반으로 퍼스트펭귄 홈페이지를 현대적이고 깔끔한 디자인으로 리빌딩했습니다.

## 📁 생성된 파일들

### 새로운 홈페이지 파일들
- `new-index.html` - Linear 디자인 시스템이 적용된 새로운 홈페이지
- `css/linear-home.css` - Linear 디자인 시스템 기반 홈페이지 스타일
- `js/linear-home.js` - 새로운 홈페이지 기능 통합 스크립트

### Linear 디자인 시스템 컴포넌트들
- `src/components/Button/` - 버튼 컴포넌트
- `src/components/Card/` - 카드 컴포넌트  
- `src/components/Navigation/` - 네비게이션 컴포넌트
- `src/components/Typography/` - 타이포그래피 컴포넌트
- `linear-theme.json` - Linear 디자인 토큰

## 🚀 적용 방법

### 1. 기존 홈페이지 백업
```bash
# 기존 홈페이지 백업
cp index.html index-backup.html
```

### 2. 새로운 홈페이지 적용
```bash
# 새로운 홈페이지로 교체
cp new-index.html index.html
```

### 3. 브라우저에서 확인
1. 브라우저에서 `index.html` 열기
2. 새로운 Linear 디자인 확인
3. 모든 기능이 정상 작동하는지 테스트

## 🎨 새로운 디자인 특징

### Linear 디자인 시스템 적용
- **색상**: Linear.app의 정확한 색상 팔레트 (#5e6ad2 브랜드 색상)
- **타이포그래피**: Inter Variable 폰트와 정확한 크기/간격
- **간격 시스템**: 일관된 패딩, 마진, 보더 반경
- **애니메이션**: 부드러운 전환 효과
- **접근성**: WCAG 가이드라인 준수

### 주요 개선사항
1. **현대적인 UI**: Linear.app의 깔끔하고 모던한 디자인
2. **일관된 컴포넌트**: 재사용 가능한 디자인 시스템
3. **향상된 사용자 경험**: 직관적인 네비게이션과 인터랙션
4. **완벽한 반응형**: 모든 화면 크기에서 최적화
5. **접근성 개선**: 키보드 네비게이션과 스크린 리더 지원

## 🐛 문제 해결

### 일반적인 문제들

1. **스타일이 적용되지 않는 경우**
   - CSS 파일 경로 확인
   - 브라우저 캐시 클리어

2. **JavaScript 오류 발생**
   - 브라우저 콘솔 확인
   - 기존 스크립트와의 충돌 검사

3. **반응형이 작동하지 않는 경우**
   - viewport 메타 태그 확인
   - CSS 미디어 쿼리 검사

### 롤백 방법
문제 발생 시 기존 홈페이지로 롤백:
```bash
cp index-backup.html index.html
```
