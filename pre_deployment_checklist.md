## 🌐 Netlify 배포 단계

### 1단계: 개발용 배포
```bash
# Netlify CLI 설치 (없는 경우)
npm install -g netlify-cli

# 로그인
netlify login

# 개발용 사이트 배포
netlify deploy --dir . --alias dev-firstpenguin
```

### 2단계: Firebase 연동 테스트
- 배포된 개발용 사이트에서 `firebase-test.html` 접속
- 모든 테스트 통과 확인
- 실제 사용자 계정으로 로그인 테스트

### 3단계: 최종 배포
```bash
# 프로덕션 배포
netlify deploy --dir . --prod
```

## 🐛 문제 해결 가이드

### 일반적인 문제들

#### 1. Firebase 연결 실패
```
문제: Firebase 초기화 오류
해결: firebase-core.js 파일의 설정 확인
```

#### 2. CORS 오류
```
문제: 도메인 허용 설정 오류
해결: Firebase Console에서 도메인 추가
```

#### 3. 권한 오류
```
문제: Firestore 보안 규칙 오류
해결: 보안 규칙 파일 확인 및 수정
```

#### 4. 캐시 문제
```
문제: 변경사항이 반영되지 않음
해결: 브라우저 캐시 클리어 또는 하드 리프레시
```
