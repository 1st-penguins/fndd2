## 2. 생활체육지도사 문제풀이 / 문제분석 점검

### 2.1 문제풀이 (연도별 기출)

- **노출 연도**: 2023, 2024, 2025만 링크 노출  
  - `index.html`에서 2022·2021은 **「준비중」** 처리 (비활성, `pointer-events: none`)
- **데이터·이미지**  
  - `data/sports/`: 2021–2025 JSON 존재  
  - `images-sports/`: **2022, 2023, 2024, 2025**만 이미지 존재 (2021은 0건)
- **시험 페이지**  
  - `exam-sports/`: **2023, 2024, 2025** HTML만 존재 (2021·2022 시험 페이지 없음)
- **결과**: 현재 사용자가 들어갈 수 있는 경로는 **2023·2024·2025** 뿐이며, 해당 연도는 이미지·데이터 모두 있어 **문제풀이는 정상 동작**합니다.

### 2.2 문제분석 (학습 분석)

- `getCurrentCertificateType()`으로 자격증별 필터링 적용
- `exam-sports/` 진입 시 `certificateType = 'sports-instructor'`로 기록 (`quiz-core.js` pathname 기준)
- `quiz-repository.js` / `quiz-data-service.js`에서 `certificateType` 저장·조회
- `analytics-dashboard.js`, `user-analytics.js`, `analytics-question-sets.js`에서 `certificateType` 기준 필터
- **결과**: 생활체육 탭 선택 시 **생활스포츠지도사 데이터만** 조회·표시되도록 구성되어 있어 **문제분석도 정상 동작**합니다.

### 2.3 2021년 이미지 누락 영향

- **현재**: 2021년 생활스포츠는 **연도 목록에 노출되지 않음** (준비중)
- **추가 시**: 나중에 `years-sports/year_2021.html` 및 `exam-sports/2021_*.html`을 넣으면, `images-sports/2021/`이 없어 **문제 이미지는 깨짐**
- **동작**: `quiz-core.js`에서 이미지 로드 실패 시 `onerror`로 **「이미지를 불러올 수 없습니다」** 문구로 대체하므로, **페이지 크래시나 빈 화면은 발생하지 않음**
- **정리**: **현재 배포 범위에서는 2021 이미지 누락이 서비스 장애로 이어지지 않음**
