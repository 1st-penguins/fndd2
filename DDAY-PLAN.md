## 1. 개요

체육지도자 자격증 시험 준비생을 위해 **시험 D-Day 카운터**와 **시험 일정 전체 페이지**를 구현합니다.

- **메인 페이지 (index.html)**: Hero 섹션과 시험 선택 사이에 D-Day 카운터 삽입
- **시험 일정 페이지 (schedule.html)**: 자격증별 전체 시험 일정을 한눈에 확인

## 4. 데이터 구조

### 4-1. Firestore 컬렉션: `examSchedules`

```javascript
// 문서 ID: 자동 생성
{
  certType: "health-manager",        // "health-manager" | "sports-instructor" | "sports-instructor-1"
  title: "필기시험",                   // 일정 제목
  type: "written-exam",              // "registration" | "written-exam" | "practical-exam" | "result" | "other"
  startDate: Timestamp,              // 시작일
  endDate: Timestamp | null,         // 종료일 (기간이 아닌 경우 null)
  year: 2026,                        // 연도 (필터/정렬용)
  half: 1,                           // 1: 상반기, 2: 하반기
  description: "",                   // 추가 설명 (선택)
  order: 1,                          // 같은 날짜 내 정렬 순서
  isActive: true,                    // 표시 여부
  createdAt: Timestamp,
  updatedAt: Timestamp
}
```

### 4-2. 일정 type 분류

| type | 한글명 | 아이콘/배지 |
|------|--------|------------|
| `registration` | 접수 기간 | 📝 |
| `written-exam` | 필기시험 | ✏️ |
| `practical-exam` | 실기/구술시험 | 🏃 |
| `result` | 합격자 발표 | 📢 |
| `other` | 기타 | 📌 |

### 4-3. 데이터 입력 방식

- **관리자 수동 등록** (Firestore 콘솔 또는 향후 관리자 페이지)
- 체육지도자 시험 일정은 연 1~2회 공지되므로 수동 관리가 적합
- 초기 데이터: 2026년 시험 일정을 시드 스크립트로 등록

## 7. D-Day 계산 로직

```javascript
function calculateDDay(targetDate) {
  const today = new Date();
  today.setHours(0, 0, 0, 0);
  const target = new Date(targetDate);
  target.setHours(0, 0, 0, 0);
  
  const diff = Math.ceil((target - today) / (1000 * 60 * 60 * 24));
  
  if (diff > 0) return `D-${diff}`;
  if (diff === 0) return 'D-Day';
  return `D+${Math.abs(diff)}`;
}
```
