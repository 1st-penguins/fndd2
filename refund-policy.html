환불 및 교환 규정

퍼스트펭귄 디지털 콘텐츠 환불 및 교환 정책입니다

⚠️ 디지털 콘텐츠 특성 안내

본 서비스는 디지털 콘텐츠(PDF 자료, 동영상 강의)를 판매하는 온라인 플랫폼입니다.

디지털 콘텐츠의 특성상 다운로드 또는 스트리밍 후에는 원칙적으로 환불이 불가능합니다.

전자상거래법 제17조에 따라 디지털 콘텐츠는 소비자가 콘텐츠를 다운로드하거나 시청한 경우 청약철회가 제한될 수 있습니다.

📄 PDF 자료 환불 정책

✅ 환불 가능 (구매일로부터 7일 이내)
  • PDF 다운로드를 하지 않은 경우 → 전액 환불
  • 다운로드 버튼을 클릭하지 않았거나 다운로드 기록이 없는 경우
  • 시스템에서 자동으로 다운로드 여부를 확인합니다
❌ 환불 불가
  • PDF 다운로드를 완료한 경우 → 환불 불가
  • 단, 다운로드한 파일이 손상되었거나 내용이 상품 설명과 현저히 다른 경우는 예외
  • 구매일로부터 7일 경과 후 환불 요청
  • 단순 변심, 학습 계획 변경 등 개인적 사유
📌 중요:

PDF는 다운로드와 동시에 복제가 가능한 특성상, 다운로드 완료 시점부터 환불이 불가합니다. 구매 전 상품 설명과 미리보기(제공 시)를 충분히 확인해주세요.

🎬 온라인 강의 환불 정책

✅ 전액 환불 가능 (구매일로부터 7일 이내)
  • 강의를 전혀 시청하지 않은 경우 → 전액 환불
  • 강의 페이지 접속만 하고 재생을 시작하지 않은 경우 포함
  • 시스템에서 자동으로 시청 기록을 확인합니다
⚠️ 부분 환불 가능 (구매일로부터 7일 이내)
  • 전체 강의의 30% 미만 시청한 경우 → 50% 환불
  • 예: 10강 중 3강 미만 시청 시
  • 구매 금액의 50%를 환불하며, 환불 처리 후 강의 접근 불가
  • 단, 쿠폰을 사용한 경우 할인된 금액 기준으로 50% 환불
❌ 환불 불가
  • 전체 강의의 30% 이상 시청한 경우 → 환불 불가
  • 예: 10강 중 3강 이상 시청 시
  • 구매일로부터 7일 경과 후 환불 요청
  • 단순 변심, 학습 계획 변경 등 개인적 사유
  • 강의 내용이 어렵다는 이유 (난이도는 주관적 판단)
📌 강의 시청률 기준:
시청률 환불 가능 여부 환불 금액
0% (미시청) ✅ 가능 100% 전액
1% ~ 29% ⚠️ 부분 가능 50% 환불
30% 이상 ❌ 불가 환불 불가

✅ 공통 환불 가능 조건 (제품 불량/오류)

다음의 경우 콘텐츠 사용 여부와 관계없이 전액 환불이 가능합니다:

1. 서비스 장애 또는 오류 발생 시

  • PDF 파일이 손상되어 열리지 않거나 내용이 깨지는 경우
  • 동영상이 재생되지 않거나 심각한 품질 저하가 있는 경우
  • 상품 설명과 실제 내용이 현저히 다른 경우
  • 서비스 제공자의 책임 있는 사유로 콘텐츠 이용이 불가능한 경우

※ 확인 후 전액 환불 또는 동일 가치의 다른 상품으로 대체 제공

2. 중복 결제 시

  • 동일 상품을 실수로 중복 결제한 경우
  • 시스템 오류로 인한 중복 결제
  • 중복 결제 금액 즉시 환불 (확인 후 1~3 영업일)

3. 오배송 (잘못된 상품 제공)

  • 구매한 상품과 다른 상품이 제공된 경우
  • 전액 환불 또는 올바른 상품으로 교체

📝 환불 신청 방법

환불 신청 절차
  1. 환불 신청: 이메일 또는 카카오톡으로 환불 요청
  2. 정보 제출: 구매 정보 및 환불 사유 작성
  3. 사용 여부 확인: 시스템에서 자동으로 다운로드/시청 기록 확인 (1~2 영업일)
  4. 환불 승인: 환불 가능 여부 검토 및 안내
  5. 환불 처리: 결제 수단으로 환불 (승인 후 3~5 영업일)

  • 필수: 환불 사유 (구체적으로 작성)

  • 시스템에서 자동으로 다운로드/시청 기록을 확인하므로, 실제 사용 여부가 확인됩니다

계좌이체 승인 후 3~5 영업일 지정 계좌로 입금
간편결제
(카카오페이, 네이버페이 등)		 승인 후 3~7 영업일 각 결제사 정책 따름
📌 환불 처리 단계별 소요 시간:
  • 1단계: 환불 신청 → 사용 여부 확인 (1~2 영업일)
  • 2단계: 환불 승인 → 결제사에 취소 요청 (즉시)
  • 3단계: 결제사 처리 → 환불 완료 (3~5 영업일)

※ 총 소요 기간: 환불 신청 후 최대 7 영업일 (평일 기준)

* 카드사 및 은행 사정에 따라 환불 처리가 지연될 수 있습니다.
* 영업일 기준이며, 주말 및 공휴일은 제외됩니다.
* 환불 진행 상황은 이메일로 안내해드립니다.

🎟️ 쿠폰 사용 시 환불 정책

쿠폰을 사용하여 할인받고 구매한 경우:

전액 환불 시 (미사용 상태)

  • 환불 금액: 실제 결제 금액 (할인된 금액) 전액 환불
  • 쿠폰 복구: 사용한 쿠폰이 1회용인 경우 재사용 가능하도록 복구
  • 예: 50,000원 상품을 10,000원 쿠폰으로 40,000원에 구매 → 40,000원 환불 + 쿠폰 복구

부분 환불 시 (강의 30% 미만 시청)

  • 환불 금액: 실제 결제 금액의 50% 환불
  • 쿠폰 복구: 불가능 (부분 사용으로 간주)
  • 예: 50,000원 상품을 10,000원 쿠폰으로 40,000원에 구매 → 20,000원 환불 (쿠폰 복구 안됨)
⚠️ 쿠폰 관련 주의사항:
  • 환불 시 쿠폰 할인 금액은 환불되지 않습니다 (쿠폰 복구로 대체)
  • 부분 환불의 경우 쿠폰이 복구되지 않으므로 신중하게 결정해주세요
  • 기간 만료된 쿠폰은 복구되더라도 사용할 수 없습니다

🔄 교환 정책

디지털 콘텐츠 특성상 물리적 교환은 불가능합니다.

교환 가능한 경우

  • 오배송 (잘못된 상품 제공): 구매한 상품과 다른 상품이 제공된 경우
  • 상품 하자: 파일 손상, 재생 불가 등 서비스 제공자 책임인 경우
  • 동일 가치 또는 더 높은 가치의 다른 상품으로 교환 가능

교환 불가능한 경우

  • 단순 변심 또는 구매 착오
  • 이미 다운로드하거나 시청한 콘텐츠
  • 구매일로부터 7일 경과 후 교환 요청

※ 교환이 불가능한 경우 환불 정책에 따라 환불 처리됩니다.
